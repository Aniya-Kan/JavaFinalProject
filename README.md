Final Project Description: Ticket Booking System
1. About the Project
Our final project is a Ticket Booking System. In this system, users can search for tickets, see ticket information, and book them. The project is simple and easy to use. Users can book tickets for transport (bus, train, or airplane).

2. Technologies (Stack)
Java — main programming language
MySQL — database for saving information
JavaFX— for the user interface

3. Database
We use MySQL to store data.
The database has several tables:
Users — information about users
Routes — destinations or travel routes
Tickets — ticket price, date, time
Reservations — user bookings
Tables are connected with primary keys and foreign keys.

4. UI Design
The user interface is simple and clear.
It has these pages:
Main page
Search page
Ticket info page
Booking page
Admin page
The design is clean and easy to understand.

5. Features
User registration and login
Search for tickets
View ticket details
Book tickets
Admin can add or edit tickets
